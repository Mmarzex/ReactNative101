Prerequisites

To make the workshop run smoothly please make sure you have the following things setup. I'm only going to being running through running on iOS but all the code is agnostic to platform so you can run it on android if you so desire.

  1. Xcode, you can download it from the Mac App Store
  2. Nodejs, Install Node
  3. Install Watchman through homebrew ($brew install watchman, If you don't have homebrew installed go to brew.sh and follow directions
  4. Install react-native through npm npm i react-native-cli -g If you get an error when installing run with sudo instead, sudo npm i react-native-cli -g
  5. You are ready!
